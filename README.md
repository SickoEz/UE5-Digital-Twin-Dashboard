# UE5 Digital Twin Dashboard

An interactive digital twin visualization project developed with Unreal Engine 5.

## Overview

This project demonstrates how Unreal Engine can be used to build a modern digital twin dashboard with real-time 3D visualization, interactive UI, and data-driven panels.

The system combines a virtual campus scene with dashboard widgets to display operational information such as:

* Campus population
* Dormitory access
* Vehicle statistics
* Security monitoring
* Device status
* Environmental data

The goal is to provide a scalable framework for digital campus visualization and smart city applications.

---

## Features

* Real-time 3D scene
* Interactive UI (UMG)
* Data visualization dashboard
* Camera switching
* Dynamic statistic panels
* Modular Blueprint architecture
* Easy integration with external data sources
